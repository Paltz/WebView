# How to run the app

1. Download Expo Go From android | ios

2. clone this repository

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
    npx expo start
   ```

5. Scan the QR code from the Expo Go app if its Android | use camera to scan the QR code if its IOS
